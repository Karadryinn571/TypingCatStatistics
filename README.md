# Typing Cat Statistics
Расширение для Visual Studio, которое показывает статистику вашей печати и маленького кота.

<br/><br/>

## Выполнил:
## Голиков Глеб Васильевич M3120

<br/><br/>

## Функции
 - Вывод скорости вашей печати в двух форматах: символы/секунда и символы/минута
 - Индикатор нажатия на клавиши: кот бьет по столу каждый раз когда вы вводите символ
 - Индикатор вашей печати: показывет когда вы печатете непрерывно
